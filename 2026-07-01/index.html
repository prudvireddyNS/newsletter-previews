Microsoft research warns poisoned MCP tool descriptions can exfiltrate data through agents Microsoft research described how attackers can hide instructions inside Model Context Protocol tool descriptions so an AI agent leaks data while each individual step still appears authorized. In Microsoft’s invoice example, a third-party enrichment tool secretly tells the agent to collect the last 30 unpaid invoices and attach them to the next tool call, leading Microsoft to recommend treating tool descriptions like system prompts, allowlisting publishers, reviewing description changes, and adding human approval for risky actions.
GuardFall bypass hits 10 of 11 open-source AI coding agents tested Adversa AI’s GuardFall research found shell-command safety checks in 10 of 11 open-source coding and computer-use agents could be bypassed with decades-old bash parsing tricks such as r''m, base64 piping, or destructive flags on ordinary tools. The tested vulnerable tools included opencode, Goose, Cline, Roo-Code, Aider, Plandex, Open Interpreter, OpenHands, SWE-agent, and Hermes, while Continue’s shell-aware parser held in its default editor mode.
