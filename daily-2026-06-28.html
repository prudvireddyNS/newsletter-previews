OpenAI previews GPT-5.6 Sol with restricted cyber-focused rollout

OpenAI previewed three GPT-5.6 variants—Sol, Terra and Luna—to a small set of government-approved companies, with broader availability planned in the coming weeks and a July Cerebras launch mentioned by Andrew Feldman. Sol is positioned as the flagship cyber-capable model with stronger safeguards for sensitive cyber requests, ExploitBench performance competitive with Anthropic Mythos Preview using about one-third the output tokens, and VulnLMP results showing credible memory-safety leads but not autonomous end-to-end attacks.

Anthropic cleared to partially release Claude Mythos 5 to trusted U.S. organizations

Reuters reported that the U.S. government is allowing Anthropic to release its powerful Claude Mythos 5 model to some “trusted” U.S. organizations. The update points to a controlled availability path for a frontier Claude model rather than a broad public launch.

Ornith-1.0 coding models land in Ollama with MIT-licensed open weights

Ollama posted run commands for Ornith, including `ollama run ornith` and coding use through Claude or Pi, after the Ornith-1.0 family was described as 9B Dense, 31B Dense, 35B MoE and 397B MoE agentic coding models released under the MIT license. The quoted release claims strong coding results including Terminal-Bench 2.1 at 77.5 and SWE-Bench scores of 82.4 verified, 62.2 pro and 78.9 multilingual, while a local test reported the 35B MoE at FP8 on one DGX Spark with a 3M-token context headroom and about 36 tok/s before speculative decoding.

OpenRouter names four open-weight models now powering agentic pipelines

OpenRouter’s June 27 insights post highlights DeepSeek V4 Flash, GLM 5.2, MiniMax M3 and NVIDIA Nemotron 3 Ultra as the open-weight models it sees mattering most for production agentic workloads in June 2026. The post says DeepSeek V4 Flash is an MIT-licensed roughly 284B-parameter MoE with about 13B active parameters, a 1M-token context, 79.0% on SWE-bench Verified and first-party pricing of $0.14 input and $0.28 output per million tokens, with caveats around China routing and data training on the first-party API.

Fugu is being positioned as frontier capability without export-control risk, while 360’s tools target automated vulnerability discovery and cyber defense or incident response.

A post on X says the mystery “Owl Alpha” model on OpenRouter is Meituan’s LongCat-2.0. The model is described as a 1.6T-total-parameter system with about 48B parameters activated on average and optimization for multi-step reasoning and tool calling.

Microsoft open-sources bitnet-cpp for 1-bit CPU inference

A post on X says Microsoft open-sourced bitnet-cpp, a 1-bit inference framework for running very large language models on CPUs without a GPU. The post claims the framework can run 100B-parameter models locally with 6.17x faster inference and 82.2% less energy use.

DGX Spark community guide shows small-cluster inference setup for large local models

A detailed DGX Spark guide describes NVIDIA’s $4,699 GB10 Grace Blackwell desktop with 128GB unified memory, 4TB NVMe storage, ConnectX-7 200GbE QSFP networking and 1 PFLOPS FP4 performance. The writeup covers a 2x Spark direct-QSFP cluster, netplan and SSH setup, NCCL checks, vLLM Docker commands, and reported model runs such as DeepSeek V4 Flash at 1M context and 40–45 tok/s on one session across two Sparks.

It claims native support for Claude Code, GitHub Copilot, Gemini CLI, Cursor and OpenCode, plus one-command installs, environment auto-detection, conversion scripts and 50K stars with 7.5K forks.

The walkthrough targets intermediate Python developers, estimates 45 minutes and less than $2 in resources, and ends with local testing plus deployment to Agent Engine on Cloud Run.
