Zackriya-Solutions/meetily

Meetily is a self-hosted, local-first AI meeting assistant for macOS and Windows, built in Rust with live Parakeet/Whisper transcription, speaker diarization, and Ollama-based summarization. It is useful for teams that want meeting notes without sending audio to a cloud service.

TryCase

TryCase provides disposable test environments for AI coding agents. The core use case is giving agent-generated code a safe, isolated place to run and validate changes before they touch shared development or production systems.

producthunt.com
testing · coding-agents

T3MP3ST

T3MP3ST is a multi-agent offensive-security framework that connects to Claude Code, Codex, Hermes, or local models and runs authorized recon, exploit, and reporting workflows. Its README claims a 90.1% pass@1 result on XBOW's 104-challenge suite, with reproducible benchmark scripts.

x.com
security · agents

Hermes MoA 2.0

Nous Research released Hermes Mixture of Agents 2.0, an open-source ensemble layer that queries models such as GPT, Claude, and DeepSeek in parallel and synthesizes a final answer. The useful signal is less another base model and more a configurable orchestration layer for reasoning, coding, and instruction-following tasks.

x.com
model-orchestration · agents

mails

mails is email infrastructure for agents, with a CLI/SDK for sending, receiving, searching inboxes, handling attachments, and extracting verification codes. It can use hosted @mails.dev mailboxes or a self-hosted Cloudflare Worker setup.
