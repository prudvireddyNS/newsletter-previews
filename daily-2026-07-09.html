HalluSquatting can steer AI coding assistants into installing attacker-controlled packages

New research describes an attack that registers plausible package or plugin names that AI assistants hallucinate, then uses prompt injection in fetched content to get attacker code run on a user machine. The researchers frame the pattern as a path to botnet-scale compromise when agents can fetch, install, and execute with little review.

OpenAI released GPT-Live-1 and GPT-Live-1 mini, voice models that can listen and speak at the same time for interruptions, silence handling, and live translation. GPT-Live-1 mini replaces the current Advanced Voice Mode by default, while paid users can access the larger model.

SpaceXAI releases Grok 4.5 with Cursor-trained coding and agent focus

SpaceXAI released Grok 4.5, positioning it as a token-efficient model for coding, app-building, research, writing, and routine knowledge work. Cursor says the model was trained jointly with SpaceXAI on developer-agent interaction data and ships with additional cybersecurity safeguards.

AI coding agents are triggering endpoint security rules built for intruders

Sophos telemetry found Claude Code, Cursor, and OpenAI Codex activity setting off behavioral detections for credential access and execution, including DPAPI browser credential reads, Windows credential-store enumeration, system-tool downloads, and startup-folder writes. The agents were not malicious, but their normal work resembled attacker tradecraft.

thehackernews.com

AMD, OpenAI, and Microsoft contribute MRC AI networking work to OCP

AMD said it is contributing MRC to the Open Compute Project with OpenAI and Microsoft, targeting scalability, resilience, and performance for large AI clusters. The company says the work is already deployed at scale in test clusters.

x.com

GitHub ‘Verified’ commits can be replayed under new hashes without breaking signatures

New research shows a signed Git commit can be rewritten into a different hash while preserving identical files, author, date, and GitHub’s “Verified” status. The issue does not let attackers change signed code, but it weakens blocklists, provenance logs, and deduplication systems that key only on verified commit hashes.

thehackernews.com

Anthropic and AE Studio propose GRAM modules as an off switch for dual-use model knowledge

Anthropic and AE Studio described GRAM, a preliminary training method that routes dual-use knowledge such as virology or cybersecurity into removable Transformer modules. Anthropic says GRAM has not been applied to production models and may never be.

x.com
