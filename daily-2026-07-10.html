Friendly Fire attack turns Claude Code and Codex security reviews into code execution

A proof-of-concept called Friendly Fire shows Claude Code and OpenAI Codex can be tricked into executing attacker-controlled binaries while reviewing untrusted code in autonomous modes. The attack hides prompt injections and execution bait in repository documentation and files rather than using plugins, hooks, MCP servers, or config changes.

thehackernews.com

npm 12 disables install scripts by default

GitHub released npm 12 with dependency lifecycle scripts, git dependencies, and remote URL dependencies made opt-in by default. The release also starts tightening granular access tokens, with 2FA-bypass token restrictions planned from August 2026 and direct publishing changes scheduled for January 2027.

thehackernews.com

Google releases Gemma 4 open models for local multimodal and agentic apps

Google announced Gemma 4, an open model family with E2B, E4B, 12B, 26B, and 31B variants. The linked model page describes multimodal reasoning, function calling for agents, support for 140 languages, fine-tuning, and efficient local deployment on edge devices and consumer GPUs.

x.com

Meta launches Muse Spark 1.1 for agentic coding and computer-use workloads

Meta launched Muse Spark 1.1, a multimodal model aimed at agentic coding, multistep reasoning, computer use, bug fixing, and large code migrations. TechCrunch reports pricing of $1.25 per million input tokens and $4.25 per million output tokens.

Simon Willison breaks down the GPT-5.6 family

Willison gives the developer view of GPT-5.6: Luna/Terra/Sol pricing, 1M context, 128K output, benchmark caveats, and new API features such as programmatic tool calling, subagents, prompt cache breakpoints, and original-detail image inputs. Useful if you need to budget and benchmark the new models rather than just read launch claims.

Deep Agents Code on NVIDIA NemoClaw

LangChain shows a governed coding-agent blueprint using Deep Agents Code, NVIDIA NemoClaw, Nemotron 3 Ultra, deny-by-default networking, human approvals, and audit logs. The practical takeaway is how to run code agents near sensitive repositories without treating the model API as the only control surface.

langchain.com
article

NVIDIA’s iterative recipe for synthetic financial datasets

NVIDIA walks through a generation-deduplication loop that produced 502,536 unique financial news headlines across 13 categories using NeMo Data Designer, NeMo Curator, and Nemotron served via vLLM. The useful pattern is global semantic deduplication, farthest-from-centroid few-shot steering, and category reweighting for domain datasets that otherwise collapse into duplicates.

developer.nvidia.com
article

Aurora 1.5 extends Microsoft’s open Earth-system foundation model

Microsoft Research describes Aurora 1.5, adding 22 weather variables, hourly temporal resolution, and probabilistic ensemble forecasts to its open Earth-system foundation model. This matters for teams building weather, energy, transport, agriculture, and climate-risk products that need an extensible model rather than a closed forecast feed.

microsoft.com
article

Apple studies self-reflective program search for long-context reasoning

Apple’s SRLM paper augments recursive long-context methods with uncertainty signals—self-consistency, trace length, and verbalized confidence—to choose better context-interaction programs. It is relevant for RAG and agent systems where simply stuffing more context does not reliably produce better reasoning.

machinelearning.apple.com
article

Google SensorFM turns wearable streams into a health foundation model

Google Research introduces SensorFM, a wearable-health foundation model pretrained on more than one trillion minutes of sensor data from five million people. The post is useful for health-AI teams thinking about time-series foundation models, label-efficient adaptation, data infilling, and grounding personal health agents in longitudinal sensor data.

research.google
