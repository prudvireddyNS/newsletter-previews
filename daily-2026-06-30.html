Meituan releases LongCat-2.0 for agentic coding

Meituan announced LongCat-2.0, the full model behind Owl Alpha on OpenRouter. The model is described as a 1.6T-parameter MoE with about 48B active parameters, 1M context, LongCat Sparse Attention, zero-compute experts, and expert groups for agent, reasoning, and interaction tasks.

Claude models are generally available in Microsoft Foundry

Anthropic says Claude models are now generally available in Microsoft Foundry, hosted on Azure and running on NVIDIA GB300 infrastructure. Claude Opus 4.8 and Claude Haiku 4.5 are available through the Messages API with Azure-native authentication, billing, networking, governance controls, and a US data zone option.

claude.com

Vercel AI Gateway adds xAI Grok audio models

Vercel announced xAI Grok realtime voice, text-to-speech, and speech-to-text models on AI Gateway with AI SDK 7 support. The launch includes xai/grok-voice-think-fast-1.0, xai/grok-tts, and xai/grok-stt with Gateway routing, observability, and spend controls.

vercel.com

X launches hosted MCP for the X API and developer docs

X introduced hosted MCP servers for the X API and X developer docs. MCP-compatible clients such as Grok Build, Cursor, Claude Desktop, VS Code, and others can connect to search posts, look up users, access timelines, fetch trends and news, manage bookmarks, and read docs through hosted endpoints.

x.com

Cursor launches native iOS app for coding agents

Cursor introduced an iOS app in public beta for paid plans. Developers can launch always-on cloud agents, remotely control agents running on their computer, review artifacts and diffs, receive notifications, and merge PRs from mobile.

cursor.com

Base44 starts rolling out Base1, its own app-building model

TechCrunch reports that Wix-owned vibe-coding platform Base44 has begun rolling out Base1, an AI model trained and optimized to create web applications from natural language using data generated from tens of millions of real user interactions on the platform.

techcrunch.com

Malicious Perplexity-branded Chrome extension intercepted searches

Microsoft found and disclosed a malicious Chrome extension named "Search for perplexity ai" that posed as an AI search tool, changed the browser search provider, routed searches and address-bar suggestions through an attacker-controlled domain, and logged queries before redirecting users to real results.

thehackernews.com
