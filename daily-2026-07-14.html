Google and Microsoft remove ModHeader after dormant browsing-history collector is found

Google and Microsoft pulled the ModHeader browser extension, used by roughly 1.6 million people across Chrome and Edge, after researchers found code that could fingerprint devices, locally collect visited domains, encrypt them, and upload them daily. The allow-list controlling the collector was empty and there is no evidence it collected data, but the upload pipeline was already shipped in the official extension.

04

MemGhost research shows one email can inject false persistent memories into AI agents

Researchers demonstrated stealth memory injection against inbox-connected agents: a malicious email can induce an agent to write a false fact into persistent memory without exposing that action in its visible response. The study used OpenClaw, whose standing instructions and user memory files are loaded into context at the start of later sessions, making memory stores a new prompt-injection boundary.
