SGLang v0.5.15

SGLang’s latest release focuses on production serving for GLM-5.2 NVFP4, with the project citing 500+ tok/s/user on 8x B300 and 450 on 4x GB300 at batch size 1. It also adds support for Hunyuan 3, HRM-Text, LocateAnything-3B, Unlimited-OCR, JoyEcho, and Qwen3.6, plus native web search, breakable CUDA Graph by default, MLA decode context parallelism, and FlashInfer all-to-all for routed MoE.

Google Labs’ TypeScript repo packages Agent Skills designed for the Stitch MCP server and compatible coding agents including Antigravity, Gemini CLI, Claude Code, and Cursor. It is useful for teams trying to standardize reusable agent behavior across editors and model providers.

AG KIT

AG KIT injects a .agents configuration folder into a project with 20 specialist personas, 45 skills, and 13 slash-command workflows. Its useful patterns are coordinator mode for multi-agent orchestration, persistent project memory, and context-compression routines that reduce repeated setup across coding-agent sessions.

JustVugg/colibri

Colibri is a zero-dependency C inference engine that runs GLM-5.2’s 744B-parameter MoE by keeping the dense core resident and streaming routed experts from disk. It is not fast, but it is a useful proof point for local MoE offload, SSD caching, and memory-management experiments.

2.1k stars · local inference · MoE

Effects SDK

An SDK for adding AI video and audio effects to real-time apps. It is relevant for teams that need media enhancement features without building and serving the underlying effects pipeline themselves.
