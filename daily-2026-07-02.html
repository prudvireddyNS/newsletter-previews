Cloudflare sets a September 15 deadline for AI crawlers to split search from training and agent use

Cloudflare says that starting September 15, 2026, its default settings will block mixed-use crawlers from ad-supported pages for new Cloudflare customers, new sites from existing customers, and all existing free customers unless site owners opt otherwise. It is also evolving Pay Per Crawl into Pay Per Use with initial partners Ceramic.ai and You.com.

Cursor 3.0 patches DuneSlide zero-click sandbox escapes

Cato AI Labs disclosed two Cursor flaws, CVE-2026-50548 and CVE-2026-50549, that could let a poisoned prompt delivered through MCP-connected services or web search escape the editor sandbox and run commands on a developer machine. Versions before Cursor 3.0 are affected; the fixes shipped in Cursor 3.0.

Meta is reportedly preparing an AI compute and model cloud business

Bloomberg reports that Meta is developing plans to sell access to AI compute power and models through a cloud infrastructure business, with TechCrunch noting the initiative is reportedly called Meta Compute. Meta has committed $182.9 billion to AI infrastructure in coming years and is looking for returns beyond its own AI products.

techcrunch.com

Unpatched Argo CD repo-server flaw can expose Kubernetes clusters to takeover paths

Synacktiv disclosed an unpatched Argo CD repo-server flaw that can let an unauthenticated attacker who can reach the internal gRPC port execute code through Kustomize’s helm-command option. The report says there is no fix or CVE yet, and recommends enabling Kubernetes network policies around repo-server and Redis.

How to Use RLMs in Deep Agents

LangChain explains how Deep Agents now uses dynamic subagents and a lightweight code interpreter to run recursive, code-driven workflows over large context instead of relying on one turn-by-turn agent loop. The useful pattern: let the model write map/reduce-style orchestration code, then dispatch subagents for deterministic coverage across huge inputs.

langchain.com
article

OpenWiki: Open Source Repo Documentation for Coding Agents

OpenWiki is an open-source agent and CLI that generates a repo wiki, points AGENTS.md or CLAUDE.md at it, and can keep it current through a GitHub Action. It is a practical answer to a common coding-agent failure mode: dumping too much stale repo context into one instruction file.

langchain.com
article

The bottleneck moved to verification. So we automated that too.

Augment describes an internal Verifier agent that deploys each PR to an isolated live environment, exercises the changed behavior, and posts logs, screenshots, API responses, and limits back to reviewers. The post is a useful blueprint for moving agentic coding from “generated code” to runtime evidence.

augmentcode.com
article

Mastering Agentic Techniques: AI Agent Reinforcement Learning

NVIDIA lays out when to use prompting, SFT, DPO, RLHF, or RLVR for agent behavior, then walks through building a small GRPO loop with verifiable rewards. The practical takeaway is to start with evals and trusted verifiers before training; otherwise RL just optimizes the wrong shortcut faster.

developer.nvidia.com
article

Multi-token Residual Prediction

Modal and NYU Shanghai show how Multi-token Residual Prediction adds a small frozen-backbone module to diffusion language models to predict inter-step residuals rather than full next-step distributions. The result is a concrete inference knob: lossless speculative speedups or faster direct decoding with measurable quality tradeoffs, with code and an SGLang implementation available.

modal.com
article

Hugging Face and Cerebras bring Gemma 4 to real-time voice AI

Hugging Face details an open, cascaded speech-to-speech stack using Parakeet for ASR, Gemma 4 31B on Cerebras for inference, and Qwen3TTS for speech output. It is useful for teams building voice assistants or robots because it decomposes the latency problem into replaceable open components instead of a black-box voice API.

huggingface.co
article
